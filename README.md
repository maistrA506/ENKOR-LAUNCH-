## 💬 WhatsApp Pre-Order Integration

Our WhatsApp integration allows customers to easily pre-order products directly through their preferred messaging platform.

### How It Works
1. Customer clicks "Pre-Order via WhatsApp"
2. WhatsApp window opens with a pre-populated message
3. Customer selects product and quantity
4. Message is sent to ENKOR sales team
5. Team responds with order confirmation and payment details

### Configuration
Update your WhatsApp business number in `js/whatsapp.js`:
```javascript
const WHATSAPP_BUSINESS_NUMBER = "+your-number-here";
```
